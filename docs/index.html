Sobre

Eu acredito no poder dos dados como um agente transformador dos negócios, isso porque eu tive a oportunidade de atuar diretamente no setor de Big Data de um dos maiores bancos financeiros nacionalmente, isso mudou a minha visão da importância dos dados e a responsábilidade que devemos ter frente a eles. A partir disso fui encorajada a aprofundar os meus conhecimentos, ocasionando na realização de uma pós-graduação em Lei Geral de Proteção dos Dados. Participei de uma concorrida formação com contratação garantida em Ciência de Dados pelo Banco Safra e Ada (antiga Let's Code) , foram disponibilizadas apenas 25 vagas para 10.000 concorrentes. Após ser aprovada finalizei o curso em Ciência de Dados e comecei a atuar no setor de Big Data do Banco Safra, o qual e 2023 recebemos um prêmio internacional.

Analista de Dados

Atualmente atuo como Engenheira de Dados no setor de Big Data no Banco Safra utilizando ferramentas como Python, Sql, Airflow, Cloudera, Apache Hive, Apache Impala e Governança de Dados.
